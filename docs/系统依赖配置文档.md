## 系统架构概述

theNPC 系统由以下 4 个服务组成：

| 服务 | 语言/框架 | 端口 | 说明 |
|------|----------|------|------|
| Frontend | Vue 3 + TypeScript | 26001 | 前端 UI |
| Backend API | Python + FastAPI | 26000 | 主业务后端 |
| Claude Service | Python + FastAPI | 25999 | Claude LLM 代理服务 |
| Gemini Service | Python + FastAPI | 25998 | Gemini LLM 代理服务 (默认禁用) |

## 环境变量配置

### Backend (`/backend/.env`)

```env
# LLM 服务地址
LLM_SERVICE_URL=http://127.0.0.1:25999
GEMINI_SERVICE_URL=http://127.0.0.1:25998

# Gemini API (用于图像生成)
GEMINI_API_KEY=your_gemini_api_key
GEMINI_MODEL=gemini-pro

# 阿里云 OSS (图片存储)
ALIYUN_ACCESS_KEY_ID=your_access_key_id
ALIYUN_ACCESS_KEY_SECRET=your_access_key_secret
OSS_BUCKET_NAME=your_bucket_name
OSS_ENDPOINT=your_oss_endpoint

# 服务配置
HOST=127.0.0.1
PORT=26000
```

### Claude Service (`/claude-local-service/.env`)

```env
# Claude API 配置 (Yuntou/原生 Anthropic)
CLAUDE_API_KEY=your_claude_api_key
CLAUDE_API_URL=https://api.anthropic.com
CLAUDE_MODEL=claude-sonnet-4-5-20250929

# Azure Claude API 配置 (Azure AI Foundry)
AZURE_CLAUDE_API_KEY=your_azure_api_key
AZURE_CLAUDE_API_URL=https://your-resource.services.ai.azure.com/anthropic
AZURE_CLAUDE_MODEL=claude-sonnet-4-5

# 当前激活配置: "yuntou" 或 "azure"
ACTIVE_CONFIG=azure

# 服务配置
HOST=127.0.0.1
PORT=25999

# 其他配置
MAX_TOKENS=4096
TEMPERATURE=0.7
```

### Gemini Service (`/gemini-service/.env`)

```env
GEMINI_API_KEY=your_gemini_api_key
GEMINI_MODEL=gemini-3-pro-preview
HOST=127.0.0.1
PORT=25998
```

## 启动指南

### 方式一：使用启动脚本 (Windows)

```batch
cd theNPC
start_system.bat
```

### 方式二：手动启动各服务

```bash
# 1. 启动 Claude Service (Port 25999)
cd claude-local-service
py -3.13 server.py

# 2. 启动 Gemini Service (Port 25998) - 可选
cd gemini-service
py -3.13 server.py

# 3. 启动 Backend API (Port 26000)
cd backend
py -3.13 main.py

# 4. 启动 Frontend (Port 26001)
cd frontend
npm run dev
```

### 访问地址

- **Frontend UI**: http://localhost:26001
- **Backend API**: http://localhost:26000
- **Claude Service**: http://localhost:25999
- **Gemini Service**: http://localhost:25998

## 故障排查

### 常见问题

1. **Claude Service 500 错误**
   - 检查 `.env` 中的 `AZURE_CLAUDE_API_URL` 格式是否正确
   - 正确格式: `https://xxx.services.ai.azure.com/anthropic`
   - 错误格式: `https://xxx.openai.azure.com/...`

2. **Backend 连接 LLM 失败**
   - 确保 Claude Service 已启动且监听在 25999 端口
   - 检查防火墙设置

3. **Frontend 无法连接 Backend**
   - 确保 Backend API 监听在 26000 端口
   - 检查 CORS 配置
